## Upstream: mlx-c (Apple)

mlx-cgo is a fork of [mlx-c](https://github.com/ml-explore/mlx-c), the official C API for Apple's [MLX](https://github.com/ml-explore/mlx) framework.

The original Python code generation toolchain (~2886 lines, 9 files) was ported to Go as part of this fork.
See [ATTRIBUTION.md](ATTRIBUTION.md) for a detailed breakdown of what was ported vs. what is new.
